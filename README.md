# PokeChat
This is the repo for the generative AI question-answering chatbot on Pokemon data using retrieval-augmented generation (RAG) in 
[*Talk to your Data with Retrieval-Augmented Generation (RAG) or: How I Built a Pok√©mon Chatbot*](https://medium.com/@hannahcierlak/talk-to-your-data-with-retrieval-augmented-generation-rag-or-how-i-built-a-pok%C3%A9mon-chatbot-2beb2457851a). 

## Specs
* LLM: Mistral 7B
* embedding model: voyage 2 from Voyage AI
* vector DB: Chroma DB
* framework: LangChain
* dataset: Master Pokemon Dataset and Corpus

## Environment
Kaggle Notebooks with GPU T4 x2
